自我介绍

项目提问

  • 简历写的博客网站是你自己搭建的吗，怎么搭建的？
  • 你的博客图片比较大，用哪些技术和方法解决这个问题？
  • 图片压缩怎么做？
  • 你有没有想到图片没加载完全之前该做什么？
  • 你这个登录的背景特效怎么做的？
  • 你做项目有意思的东西是什么，学习很感兴趣的有哪些？

数据结构

  • 什么是图？
  • 什么是链表？
  • 什么是树？
  • 讲一讲贪心算法？

计算机网络

  • 讲一讲TCP/IP协议？
  • 讲一讲有几种HTTP协议？
  • 讲一讲http的传输方式？
  • 讲一讲options请求一般用在哪些情况？

团队协作

  • git返回上个版本怎么做？
  • git撤回到倒数第三个版本
  • git revert有用过吗

团队项目

  • 组件库用的是什么架构？什么是单包架构？

  • 用的什么方式暴露的？

  • 单包架构能否进行组件的单独引入？

  • 你们做的低代码平台项目的产物是什么？

  • 类似于问卷星吗？

  • 项目中使用了栈结构存储各类操作，如回退、删除、清空，你存储的每个元素具体有什么，怎么实现的？

  • 清空后的话，会不会特别不安全，有没有做什么保障？

  • 你删除后再回退，组件能出现吗？

  • 你觉得你在这个项目中感觉技术性很强的功能是什么？

杂谈

  • 你最近感觉最有成就感的事情是什么？
  • 你说了three.js，这是一个怎么样的框架？
  • 讲一讲solidjs?
  • Single你了解吗，solidjs中的一个重要概念？
  • 我看你参加了ACM，蓝桥杯，蓝桥杯比赛比什么？
  • 风车动画你怎么实现的？

算法

  • textarea中需要对表单数据进行验证，实现一个算法，验证textarea中的数据不包含单行注释：//，遇到它就进行提示，但是遇到链接如http://www.baidu.com这类情况需要进行保留。
    • （第一次我的方法）
      • 正则匹配+先行断言
    • 如果浏览器比较低，不兼容先行断言的这个正则规则，怎么做？
      • （第二次我的回答）
        • 使用include，indexof进行查找
      • 这个方法只能查找到第一个，不能查找到多个怎么办？
        • reg.match方法，通过传入回调替换掉http:，或者判断是否和单行注释的正则匹配相等

反问

  • 反馈有多久？
  • 有几面？
  • 业务是什么？低代码吗？
