这是我参与「第五届青训营 」笔记创作活动的第 3 天.

# 前言

该笔记为 JavaScript 的字节青训课程笔记，，作为 JS 的进阶学习，重点内容有五个方面：

  • 编码原则
  • 组建封装
  • 过程抽象
  • 质量优化

诸位看官可以逐情食用。

# 编码原则之各司其责

  • 各司其职：HTML，CSS，JavaScript 职能分离

    • 避免不必要的 JS 操作样式
    • 用 class 表示状态
    • 纯展示类交互寻求零 JS 方案

  • 组建封装：给封装的 UI 组件具有正确性，扩展性和复用性

  • 过程抽象：使用函数式编程思维

    • 只调用一次，闭包和高阶函数
