# JSON

  • JSON: JavaScript Object Notation JS 对象描述符
  • 前后端交互的一种数据格式，我们可以把 JSON 数据扔给后端，后端也可以响应 JSON 数据

# JSON 格式

  • 格式一：字符串

    • 不能使用单引号

    • 很少使用直接的字符串类型，而是被包装成后两种

  • 格式二：对象

  • 格式三：数组

# JSON 序列化和反序列化

  • 序列化：把对象转成 JSON 串

  • 反序列化：把 JSON 串还原成 JS 对象

  • 序列化和反序列化过程是深度拷贝
