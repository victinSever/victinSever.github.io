# OOP

# 对象的创建方法

  • new Object()
  • let obj = {}

# JS 的内置类

  • 常用的：Number，Array，Date，Function，Math，String，Boolean，Window，Set，Map
  • Math，Window 等是单体内置类（不能 new）

# JS 所有数据都是对象

因此，在开发中，一般不直接操作原生 DOM，而是使用项目框架来操作虚拟 DOM，提高性能。

  • 对象中的键都是字符串，不管加不加单双引号

  • 键也可以是其他值

  • 对象没有 length

  • 遍历对象

# 对象创建方案

  • 通过字面量的形式创建对象

    缺点：①大量重复代码；②内存空间浪费

  • 工厂函数创建对象

    直接产出对象

    缺点：获取工厂化对象时会通过字面量形式

  • 构造器创建对象

    缺点：也会造成空间浪费

  • 构造器 + 原型对象创建对象

推荐使用：构造器 + 原型对象
